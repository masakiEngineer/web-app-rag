【タイトル】VPN接続の設定方法（Windows）

【内容】
1. Windowsの「設定」から「ネットワークとインターネット」→「VPN」を選択します。
2. 「VPN接続を追加する」をクリックし、以下の情報を入力してください。
   - 接続名：会社VPN
   - サーバー名またはアドレス：vpn.example.com
   - VPNの種類：L2TP/IPsec
   - サインイン情報の種類：ユーザー名とパスワード
3. 入力後、「保存」をクリックし、「接続」ボタンでVPNに接続できます。

【注意事項】
VPN接続には事前に配布された証明書が必要です。インストールしてから接続を試みてください。

【タイトル】アカウントがロックされた場合の対応

【内容】
Active Directoryアカウントがロックされる主な原因は、複数回の誤ったパスワード入力です。
ロックされた場合は、30分経過で自動解除されるか、社内ITサポートに連絡してください。

【連絡先】
it-support@example.com または 内線1234

【タイトル】メールの送信ができない場合の対処法

【内容】
Outlookでメール送信ができない原因は主に以下のとおりです：
- インターネット接続の問題
- SMTPサーバの設定ミス
- メールボックスの容量制限

【対処方法】
1. ネットワーク接続を確認
2. 設定 → アカウント → SMTPサーバ設定を見直す
3. ゴミ箱や不要メールを削除し、容量を確保する

【タイトル】Teamsの通知が来ない

【内容】
通知が来ない主な原因は以下のとおりです：
- 通知がオフになっている
- Windowsの集中モードがオン
- モバイルアプリの設定ミス

【解決方法】
1. Teamsの設定 → 通知 で確認
2. Windows設定 → 集中モードをオフ
3. スマートフォンの設定でTeamsの通知を許可
